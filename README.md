# GrowtopiaScripts
Growtopia Python Scripts

BEFORE RUNNING ANY OF THE SCRIPTS, MAKE SURE TO DOWNLOAD THE LIBRARIES!!

RUN THESE COMMANDS IN CMD OR POWERSHELL:

pip install PyAutoGUI

pip install keyboard

pip install pywin32
